What is etcd?

etcd is a distributed, reliable key-value store for critical data in distributed systems.

Key-Value Store Basics

etcd stores data as key-value pairs, supporting hierarchical keys and prefix queries.

Consistency & CAP Theorem

etcd is CP (Consistency + Partition tolerance) in the CAP theorem.

Raft Consensus Algorithm

etcd uses Raft to ensure all nodes agree on the same data.

Core Operations

PUT, GET, DELETE, and more.
